# API Security Risk Analysis

This project analyzes security risks in public APIs using Postman.

## Tools Used
- Postman
- JSONPlaceholder API
- ReqRes API

## Scope
- Read-only testing of public/demo APIs
- No exploitation or harmful activity

## Key Findings
- Lack of authentication (partial)
- Information disclosure in error messages
- Missing rate limiting
- Excessive data exposure
